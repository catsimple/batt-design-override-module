# Release v2.0.7

## 更新日志

### ✨ 新特性 (Features)
*   **智能内核模块识别**：
    *   新增对 `androidXX-kernel_module.ko` 命名格式的支持。
    *   优化了模块下载逻辑，优先匹配当前设备的 Android 版本，提高兼容性。
*   **沉浸式体验**：
    *   全面启用 Edge-to-Edge (沉浸式状态栏) 效果，界面更现代。
    *   修复了顶部标签栏与状态栏重叠的问题。
*   **动态界面**：
    *   **智能标签页**：未安装的模块（电池/充电）将自动隐藏对应的标签页，界面更简洁。
    *   **Release 优化**：正式版默认隐藏“状态”调试页，仅保留核心功能。

### 🛠️ 修复与优化 (Fixes & Improvements)
*   修复了构建脚本中的代理连接问题。
*   优化了模块加载时的版本回退机制。
